## Installation
```bash
npm install -g alfred-nym
```

## Getting started
Run `nym` to get **synonyms**. Pass the `-i` flag to get **antonyms**:
- Run `nym happy` to get synonyms of *happy*.

- Run `nym happy -i` to get antonyms of *happy*.
